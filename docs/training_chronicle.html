KEY FINDING
Fine-tuning on 12 examples embedded project-specific knowledge (version bugs, locked parameters, confounding patterns) but couldn't match Sonnet 4.6's reasoning and presentation quality. The Qwen model generates in think-mode — correct knowledge buried in unpolished, stream-of-consciousness output. Value proposition: domain knowledge complement via RAG, not full replacement.
